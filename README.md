# Donasi-Guru-Ngaji-Madrasah
Program Donasi Guru Ngaji merupakan gerakan kepedulian untuk membantu para guru ngaji, ustaz, dan ustazah yang dengan ikhlas mengabdikan diri dalam mengajarkan Al-Qur'an. Melalui donasi yang terkumpul, kami berupaya memberikan bantuan insentif dan dukungan pendidikan agar mereka dapat terus menjalankan tugas mulianya dalam membina umat.
